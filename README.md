### 🔮 Featured Projects

#### 👁️ [FaceSense AI](https://github.com/remijerphin-arch/Face.AI)
> A premium, privacy-first AI face analysis web application with a dynamic neon landmark mapping system.

- **Landmark Mapping**: Draws a 68-point dynamic mesh canvas directly over the user's face in the browser.
- **Deep Analysis**: Core logic calculates face shape classes (Oval, Square, Heart, etc.) and analyzes skin undertone metrics.
- **Grooming & Fashion**: Recommends personalized hairstyles, beard contours, clothing styles, and color palettes.
- **Interactive Chatbot**: Integrated Gemini LLM contextually loaded with the user's facial scan results for real-time consultation.
- **Stack**: Next.js, FastAPI, NumPy, PIL, Gemini API.

#### 📄 [Ultimate AI PDF Editor](https://github.com/remijerphin-arch/AI-PDF-Editor)
> A comprehensive web tool for AI-powered document manipulation, format conversion, and conversational insight extraction.

- **Structure Editing**: Supports merging, deleting, rotating, and reordering pages.
- **Advanced Export**: Converts PDFs to Word (docx), PowerPoint (pptx), Excel (xlsx), plain text, or images.
- **AI Integration**: Ask questions and chat directly with your document, summarize complex texts, or outline changes via Gemini LLM.
- **Security & Formatting**: Includes PDF compression, secure password protection, and custom watermarking.
- **Stack**: Next.js, FastAPI, pypdf, pdfplumber, openpyxl, python-docx, python-pptx, Gemini API.

#### 🌐 [Futuristic AI & ML Portfolio](https://github.com/remijerphin-arch/Personal-Portfolio)
> A Tesla & Apple-inspired glassmorphism portfolio site showcasing academic and professional history.

- **Circular Loader**: Custom boot sequence simulating memory allocations, ISRO telemetry data initialization, and loading checks.
- **Theme Matrix**: Instant multi-accent palette remapping across the entire website with 10 custom neon color presets.
- **IDE Resume**: A VS-Code style JSON code viewer displaying academic history, certifications, and project specs.
- **Stack**: React (Vite), Tailwind CSS v4, Framer Motion, Rolldown.
